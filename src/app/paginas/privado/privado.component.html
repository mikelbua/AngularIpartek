Zona Privada

Solo pueden acceder usuario logeados

Para proteger el accesoa esta RUTA devemos usar una guarda(GUARD) y definirla en routing

path: 'privado', component: PrivadoComponent , canActivate: [LoginGuard]
canActivate
Guarda para indicar si se pude activar esa RUTA.
LoginGuard
Guarda creada por nosotros qeu retornara true o false
